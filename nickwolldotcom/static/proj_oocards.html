OOCards

View Source

OOCards (Oh-Oh Cards) is a collection of very simple Java classes built to emulate a deck of cards.

The Main method generates a new deck of cards, shuffles it, and cuts it, printing the contents of the deck to console at each step.

The project is meant to demonstrate a basic proficiency in Object Oriented programming and the source is available for review at the project repository on Github.
